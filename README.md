# Система планирования и отчётности подразделений

## Быстрый старт (разработка)

### 1. Установка зависимостей (один раз)
```
pip install -r requirements.txt
```

### 2. Запуск (dev-режим)
```
python app.py
```

### 3. Открыть в браузере
```
http://localhost:5000
```

**По умолчанию:** логин `admin`, пароль `admin`

---

## Продакшн-запуск

### Запуск через waitress (рекомендуется)
```
server_start.bat
```
или вручную:
```
python -m waitress --host=0.0.0.0 --port=5000 app:app
```

### Переменные окружения (необязательно, но рекомендуется)
| Переменная | По умолчанию | Описание |
|---|---|---|
| `DATABASE_URL` | `postgresql://postgres:postgres@localhost:5432/planapp` | Строка подключения к PostgreSQL |
| `SECRET_KEY` | dev-ключ (небезопасен!) | Секретный ключ Flask-сессий |
| `DB_POOL_MAX` | `50` | Макс. соединений в пуле |
| `TASKS_MAX` | `500` | Макс. задач на странице |
| `PORT` | `5000` | Порт сервера |

---

## База данных

Приложение использует **PostgreSQL**.

### Создание БД (первый запуск)
```sql
CREATE DATABASE planapp;
```
Таблицы создаются автоматически при первом запуске.

Строка подключения задаётся через `DATABASE_URL`, либо используется значение по умолчанию:
```
postgresql://postgres:postgres@localhost:5432/planapp
```

## Возможности

- **Логин/пароль** — авторизация, управление пользователями (до 1000)
- **Календарь** — фильтр задач по году и месяцу
- **Таблица задач** — все 15 столбцов согласно ТЗ
- **Поиск** — фильтрация по всем полям
- **Выпадающие меню** — Отдел, Сектор, Проект, Этап, ФИО
- **Справочники** — управление значениями выпадающих меню
- **Окно Отчёт** — все 12 столбцов, открывается кнопкой в строке
- **Настройки ширины столбцов** — перетаскивание границ, сохраняется для каждого пользователя
- **Управление пользователями** — создание/удаление (только администратор)
- **Сжатие ответов** — flask-compress для ускорения загрузки
- **Security-заголовки** — X-Content-Type-Options, X-Frame-Options, X-XSS-Protection и др.
- **Rate limiting** — защита от брутфорса

## Для работы в сети (несколько пользователей)

Сервер запускается на `0.0.0.0:5000`, доступен по IP в сети.

- Локально: `http://localhost:5000`
- По сети: `http://103.112.171.163:5000`
- По домену: `http://mysuperplan.duckdns.org:5000`

### Обновление DuckDNS IP
```
duckdns_update.bat
```
