Последовательность процессов

  1. Портфель проектов — перечень всех проектов со сроками и объёмами работ с учётом запросов заказчиков.
  2. Балансировка предприятия (Б1) — определяется общая потребная загрузка на весь период. Загрузка предприятия — объём работ за период с учётом мощности. Если потребность > мощности — корректируем портфель.
  3. Генеральный график — по каждому проекту разрабатывается ГГ, задаются внутренние параметры (сроки). Для текущих — действующий ГГ, для перспективных — проект ГГ.
  4. Проект сквозного плана-графика — детализация ГГ с участием подразделений-исполнителей, исходя из располагаемой мощности. Проводится балансировка загрузки подразделений (Б2).
  5. Утверждение сквозного — из отбалансированного проекта формируются проекты производственных планов. Сквозной план-график выпускается. Повторная балансировка загрузки подразделений.
  6. План работ предприятия — формируется из всех выпущенных сквозных. Является средством контроля. При дисбалансе (Б3) — корректировка сквозных графиков.
  7. Планы подразделений — на основании плана работ предприятия формируются ПП для каждого подразделения.
