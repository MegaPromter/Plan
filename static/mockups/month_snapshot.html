Как это работает:
  • Клик по заголовку снимка — сворачивает/разворачивает блок. Когда свёрнут — видна только одна строка с ключевыми цифрами.
  • Клик по ряду снимка (например «Просрочено 41») — активирует чип «Просрочено» в тулбаре, таблица фильтруется.
  • Клик по чипу группы (Все / План / Долги) — переключает контекст, чипы статусов ниже подсвечивают то, что доступно в текущей группе.
  • Слева у каждой строки таблицы — цветная полоса: 🟢 зелёная для задач месяца, 🟠 оранжевая для долгов. Плюс — тултип при наведении.
